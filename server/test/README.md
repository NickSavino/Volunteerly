# Server Tests

This folder contains automated tests for backend services, middleware, and moderator-related workflows in the Volunteerly API.

These tests mainly cover:

- core service-layer business logic
- moderator ticket, volunteer, and chat flows
- middleware behavior such as authentication and role enforcement
- backend test environment setup and shared test utilities

Use this folder to verify API-side behavior, request authorization rules, and service correctness before integrating changes into the full application flow.
